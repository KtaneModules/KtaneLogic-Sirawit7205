On the Subject of Logic

Logic is easy, but bomb defusal AND logic might not.

  • Each row displays 3 letters. Each letter represents a statement.
  • On each row, solve GREEN statements first. (They are in the parentheses.)
  • Statements are joined with ∧ symbol (Conjunction, both statements must be true for the result to be true.) and ∨ symbol. (Disjunction, one of the statements must be true for the result to be true.)
  • Find the end result of each row, and then use the T/F button to the right to select True/False.
  • Press “Submit” when done.
